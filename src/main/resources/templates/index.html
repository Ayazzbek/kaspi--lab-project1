File Uploader

Профессиональный сервис для асинхронной загрузки файлов
с поддержкой идемпотентности и MinIO хранилищем

Идемпотентность

Повторные запросы не создают дубликатов. Один файл — одна загрузка.

Асинхронность

Файлы загружаются в фоне. Получайте статус в реальном времени.

Консистентность

Гарантированная целостность данных и компенсирующие транзакции.
